CIS Controls

What are CIS Controls?

The Center for Internet Security (CIS) Controls are a prioritised set of actions that provide specific and actionable ways to stop today's most dangerous attacks. Think of them as a cybersecurity to-do list ranked by what will protect you most.

Instead of trying to do everything at once, CIS Controls tell you which security measures to implement first for maximum protection with limited resources.

The Three Implementation Groups

  • Implementation Group 1 (IG1): For small businesses with limited cybersecurity expertise. Focuses on basic cyber hygiene with simple, low cost solutions.
  • Implementation Group 2 (IG2): For businesses with some IT resources. Adds more sophisticated security measures and monitoring capabilities.
  • Implementation Group 3 (IG3): For organisations with dedicated cybersecurity staff. Includes advanced threat hunting and specialised security tools.

Most small businesses should focus on IG1 first, then gradually move toward IG2 as their security maturity and resources grow.

CIS Critical Security Controls (Top 6 for Small Business)

  • CIS Control 1 - Inventory and Control of Enterprise Assets: Make a list of every computer, phone, tablet, and server in your business. Use tools like Lansweeper (free for 100 assets) or simply create an Excel spreadsheet.
  • CIS Control 2 - Inventory and Control of Software Assets: Document all installed software and remove programs you don't need. Use Windows "Programs and Features" or Mac "Applications" folder to audit monthly.
  • CIS Control 3 - Data Protection: Identify where your sensitive data lives and encrypt it. Enable BitLocker on Windows or FileVault on Mac for all business computers.
  • CIS Control 4 - Secure Configuration of Enterprise Assets and Software: Change default passwords, disable unnecessary services, and use security benchmarks. Download CIS Benchmarks for free from the CIS website.
  • CIS Control 5 - Account Management: See our Identity & Access Management guide for detailed user account control procedures.
  • CIS Control 6 - Access Control Management: Limit who can access what data and systems. Use Windows Active Directory or Google Workspace admin controls to set permissions.

Implementation Roadmap for Small Business

  • Week 1-2: Complete asset inventory using our provided spreadsheet template. Count every device that connects to your network.
  • Week 3-4: Audit and document all software. Remove unused programs and update everything to the latest versions.
  • Month 2: Implement data encryption on all devices and identify where sensitive information is stored.
  • Month 3: Download and apply CIS Benchmarks for your operating systems. Start with Windows 10/11 or macOS benchmarks.
  • Month 4: Review and clean up user accounts. Remove old accounts and limit access permissions.
  • Ongoing: Update your inventories monthly and review access permissions quarterly.

Simple Steps to Start Today

  • ☐ Download our CIS Controls checklist template and asset inventory spreadsheet
  • ☐ Walk around your office and list every computer, phone, tablet, printer, and network device
  • ☐ Check each computer's installed programs and uninstall software you don't recognise or use
  • ☐ Enable BitLocker (Windows) or FileVault (Mac) on all business computers
  • ☐ Visit the CIS website and download free security benchmarks for your operating systems
  • ☐ Review user accounts in your systems and remove access for former employees
  • ☐ Set monthly calendar reminders to update your asset and software inventories

Admin Privileges and User Controls

What Are Admin Privileges?

Admin privileges are special permissions that let someone control the computers and systems in your business. With admin rights, a person can install or delete software, change settings, and access all your data.

Because of this power, if a cyber criminal gets hold of an admin account, they can do a lot of damage very quickly, like stealing customer info, locking you out, or installing harmful software.

Why Should You Limit Admin Access?

Not everyone in your business needs admin rights. Giving these permissions to too many people increases the risk that a mistake or a cyber attack could cause serious problems.

Keeping admin access limited helps:

  • Reduce chances of cyber criminals taking control
  • Prevent accidental changes that might break your systems
  • Make it easier to spot suspicious activity

How to Control Admin Privileges

1. Find Out Who Has Admin Access

Ask your IT person, or if you don’t have one, check the user accounts on your computers and systems.

On Windows, go to Settings > Accounts > Family & other users or Control Panel > User Accounts to see who has admin rights.

On Mac, go to System Preferences > Users & Groups and look for accounts marked “Admin.”

2. Remove Admin Access from People Who Don’t Need It

Only give admin access to people who really need it, usually just you or trusted tech helpers.

Change other accounts to “standard user” so they can’t make big changes.

If you don’t know how, look for online guides or ask a trusted expert to help.

3. Use Strong Passwords and Multi-Factor Authentication (MFA) on Admin Accounts

Make sure admin accounts have strong, unique passwords that no one can guess.

Turn on MFA — this means logging in requires something extra like a code sent to your phone, so even if a password is stolen, the account stays safe.

See our MFA guide for help

4. Use Standard Accounts for Daily Work

Everyone else should use standard user accounts without admin rights for normal tasks like checking email or using software.

This helps prevent accidental changes or malware from spreading if someone clicks a bad link.

5. Remove Admin Access Immediately When Someone Leaves or Changes Role

If a staff member or contractor leaves your business, immediately remove their admin and user access.

This stops ex-employees from accessing your systems or data.

6. Monitor and Review Admin Account Use Regularly

Keep track of when admin accounts are used, and watch for anything unusual like logins at strange hours.

If you don’t have a system for this, set a calendar reminder to review admin users and passwords every 3 months.

Real World Example: What Can Go Wrong

Case: In 2021, JBS Australia, a major meat processor, was hit by ransomware because admin rights were too broadly given and not secured properly.

Impact: The ransomware shut down multiple facilities, causing millions in losses and disrupting supply chains.

Fix: After the attack, JBS tightened admin controls by limiting access, enforcing MFA, and monitoring account activity.

Source: ACSC JBS Ransomware Attack - What You Need to Know

Start Improving Your Admin Controls Today

  • ☐ Identify who currently has admin access on your computers and systems
  • ☐ Remove admin rights from anyone who doesn’t need it
  • ☐ Make sure admin accounts have strong passwords and MFA enabled
  • ☐ Set everyone else to use standard user accounts for daily tasks
  • ☐ Regularly check admin accounts and remove access immediately when people leave

Taking these steps helps protect your business from hackers and accidental mistakes that could cause big problems.
